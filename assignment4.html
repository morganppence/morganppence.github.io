Challenge 5: Accessibility Concerns

I think there could potentially be several accessibility concerns on this page. One of the biggest concerns that I think could arise, would be the way the google page would be read. Because I used a table to format the information, if the site were to be read aloud for a user with vision imparement, the information could be translated incorrectly by a computer. When doing research on acciessibility, I also came across some information that talke about how screen readers can have issues with forms due to the way they're internally formatted which could potentially happen with my site. As of now, there isn't a way to smoothly navigate back to the home page so that's also an accessibility issue.
